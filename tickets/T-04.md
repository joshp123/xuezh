    # T-04: Dataset import framework (HSK vocab/chars/grammar + frequency lists)

    ## Goal
    Import external datasets into SQLite for HSK alignment audits.

    ## Deliverables
    - `chlearn dataset import --type <hsk_vocab|hsk_chars|hsk_grammar|frequency> --path FILE`
- Store dataset version metadata and items
- Provide deterministic ordering (dataset order or frequency_rank)
- Include small fixture datasets under `tests/fixtures/` for tests


    ## Acceptance criteria
    - Import is deterministic and idempotent (same file/version doesn't duplicate)
- Report can join items to HSK level once imported


    ## Test plan (RGR)
    - E2E: import fixture dataset then query counts
- Integration: validate output envelope and schema

- See `specs/datasets-format.md` for accepted import file formats.
