    # T-11: Content cache v1 (store/retrieve generated stories/dialogues/exercises)

    ## Goal
    Cache model-generated content as primary-source artifacts; backend remains non-judgmental.

    ## Deliverables
    - `xuezh content cache put --type story|dialogue|exercise --key HASH --in FILE`
- `xuezh content cache get --type ... --key HASH`
- Store metadata in `generated_content` table


    ## Acceptance criteria
    - Cache is deterministic and idempotent per key
- No quality scoring in backend (ZFC)


    ## Test plan (RGR)
    - Integration: put/get returns same content
- Unit: path safety
